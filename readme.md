This contains the coding component for the data collection chatbot

Including the code to webscrape and parsing chatlogs of conversation (Webscraping.ipynb)
Join function to join survey and chatbot questionaires (Join.ipynb)
Send biweekly reminders to users (Email.ipynb)
Puesudo data analysis on collected data (Data analysis.py)
Powershell commands to access RedCap API (REDCap.sh)
collected survey data (amr_survey_output.csv)
Prompt when sending reminder emails to users (email prompt)
Assets folder contains images used in the UI design.
The code to run for adversarial prompt testing at the separate folder (Advtest.md)
