# Data Dictionary - S16 Example Data

Brief overview of the example datasets used in the Line Chatbot Antimicrobial Resistance (AMR) study.

## S16.1 Simulated Data.csv
Description: Simulated survey responses with symptoms, antibiotic usage, and age demographics.
Source: Generated by R script (S15 Data Generator Code.R) for testing and visualization purposes.

## S16.2 Raw Conversations.csv  
Description: Sample chat conversations between users and the AMI chatbot showing the survey flow.
Source: The raw conversation logs gathered from webscraping (S13 Webscraper Code.py)

## S16.3 Structured Conversations.csv
Description: Processed conversation data extracted from raw chats into structured format.
Source: Parsed and cleaned from S16.2 Raw Conversations.csv (S13 Webscraper Code.py)

## S16.4 Demographics.csv
Description: Participant background information including location, education, and occupation.
Source: Survey responses collected through the Line chatbot registration process (Downloaded from Microsoft Forms)

## S16.5 Merged Data.csv
Description: Combined dataset linking conversation data with demographic information.
Source: Merged from S16.3 and S16.4 using data join scripts (S14 Data Join Code.py)
